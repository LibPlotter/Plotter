Данный архив содержит каталоги зависимых проектов CMake, исходные коды и инструкции к графической библиотеке и предназначен для автоматической сборки на системе Linux.
В корневом каталоге кроме инструкций содержится файл CMake автоматической сборки библиотеки.

В каталоге автоматизированной сборки находится каталог AF03GL с необходимыми файлами сборки F03GL. В другом каталоге находится исходный код библиотеки разнесённый на два каталога - тесты и сама библиотека. После сборки исполняемые файлы помещаются в каталог bin, файлы библиотеки в каталог lib, файлы .mod в каталог inc.

Некоторые файлы необходимы для тестирования библиотеки и их использования не является обязательным. Далее перечисляются данные файлы:
./lib/libEFE*
./inc/plotter_functions.mod
Данные файлы используются при демонстрации передачи функций библиотеке и их использование не является обязательным при использовании библиотеки.

Данная бибилотека тестировалось на Ubuntu Linux 16.04.02 x64, Mint Mate 17.3.
Система сборки CMake и библиотека тестировалась на Ubuntu 18.04.2 LTS x64 WSL

Требуется компилятор C(C++) и Fortran. Дополнительно требуется библиотека freeglut3. Дополнительно требуется система автоматизации сборки приложений CMake.
Версии gfortran и gcc - устанавливаемые по умолчанию. Специфичная версия не применялась, проверено на 4.*.* и 7.3.0.
Также можно использовать Intel Compilers, проверено с версии XE 17.

Установка требуемых зависимостей:
sudo apt-get install libxmu-dev libxi-dev
sudo apt-get install freeglut3 freeglut3-dev
sudo apt-get install cmake
sudo apt-get install gfortran

Далее приведены электронные ресурсы для зависимостей.
Сайт F03GL: http://www-stone.ch.cam.ac.uk/pub/f03gl/index.xhtml
Примечание:
В каталоге автосборки AF03GL находится урезанный вариант библиотеки F03GL. В ней только все необходимые элементы для компиляции библиотеки. 
Сайт CMake: https://cmake.org/

Для сборки используются следующие команды, их нужно исполнять из каталога с файлом CMakeLists.txt:
	1.Установить зависимости
	2.Перейти в корневой каталог с CMakeLists.txt. Предполагается что структура файлов архива не изменялась.
	3. cmake .
	Точка обязательна
	4. make all
Данные команды собирают библиотеки и тестовые примеры и перемещают в каталоги упомянутые выше.
